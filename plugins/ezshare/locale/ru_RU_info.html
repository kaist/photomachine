Плагин "ezShare"

Плагин позволяет получать фотографии через WiFi с карты памяти ezShare (со встроенным WiFi модулем)
Поиск и скачивание фотографии ведется по всем папкам и только jpg файлы. Фотографиий, которые уже скачались, не скачиваются повторно - за это отвечает втроенная база данных. В настройках, можно включить автоматическую очистку этой базы данных при старте, либо сделать это вручную.
Плагин подключается к карте памяти по WiFi с периодичность, которая задается в настройках. Вы можете продолжать фотографировать, плагин будет скачивать новые фотографии автоматически.

Только для Windows: Вы можете включить автоматическое подключение к WiFi сети ezShare. Для этого подключитесь к ней один раз стандартными средствами windows. После этого, выберите эту сеть в настройках.
Так же, вы можете включить автоматическое переключение на другую WiFi сеть, после того как плагин закончит свою работу. Это бывает полезно, если эти фотографии в дальнейшем нужно загрузить куда то в интернет. Не забудьте поставить паузу между подключениями побольше!

Полезно: При подключении к WiFi сети ezShare, автоматически запускается браузер со страницей ezShare. Если это вас тоже бесит как и меня, нажмите на волшебную кнопку, которая отключает в Windows такое поведение. Не забудьте потом включить обратно.
