Plugin "in GoogleDrive"

The plugin uploads photos to the specified Google Drive folder.

To get started, log in using the "log in" button. A browser window will open where you need to log in to Google and allow the application to access the data. After authorization, return to the application and you will see that your authorization data has been successfully recorded in the settings.

Enter the link to the google drive folder and save the settings. The link has the form: https://drive.google.com/drive/folders/1-XweWe55-uIfG916QwzFEzH

It is very important that you have editing access in this folder!
