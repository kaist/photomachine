Плагин "Сохранить оригинал"

Плагин сохраняет оригинальный (исходный) файл в выбранную папку. Может быть полезно, например, если фотомашина используется только для фильтрации или сортировки фото.

ВНИМАНИЕ!!! плагин может сохранить оригинал только фотографий, которые получены с диска, но не фотографий, которые получены из сети (например из телеграма или WiFi)
