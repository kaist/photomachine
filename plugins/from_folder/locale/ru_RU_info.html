Плагин "из папки"

Плагин загружает фотографии из папки. Можно сделать это один раз, либо сканировать папку с некоторой периодичностью, если это задано в настройках. Эта опция полезна, если в компьютер вставляет флешка с фотографиями и нужно автоматически брать от туда новые фотографии. Настройки:
  • Искать в подкаталогах - если установлено, то плагин будет сканировать фотографии не только из этой папки, но еще и во всех ее подкаталогах.
  • Не загружать фотографии, только метаданные - включите если после этого плагина идет фильтр (например по рейтингу). Это ускорит загрузку фотографий. Не забудьте потом добавить плагин "загрузить исходное"
  • Не открывать фотографии, которые уже были обработаны. Если установлено, то плагин запоминает уже открытые фотографии с своб базу данных и не обрабатывает их повторно.
  • Очищать базу данных при старте. Если установлены, то при старте будет очищена база данных обработанных фотографий. Так же тут можно очистить ее вручную.
  • Прочитать папку один раз, либо сканировать с переодичность, которая задается в настройках.
  • Удалять исходные. Нужно в редких случаях, я бы не советовал ставить галочку тут.
